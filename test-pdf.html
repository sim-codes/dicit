üîß Troubleshooting

If the extension doesn't work with PDFs:

  1. Check File URL Permission: Go to chrome://extensions/ (Chrome) or edge://extensions/ (Edge), find Dicit, click "Details", and enable "Allow access to file URLs"
  2. Reload the Extension: In the extensions page, click the reload icon for Dicit
  3. Restart Browser: Close all browser windows and restart Chrome/Edge
  4. Check Console: Press F12, go to Console tab, and look for any Dicit-related messages
